Simple Flexbox Grid

A simple responsive grid system using flexbox for layout. Kind of works in IE11 but who cares about that anymore right?

This markup:

<div class="grid">
        <div class="row">
            <div class="cell">Auto</div>
            <div class="cell half">Half</div>
            <div class="cell">Auto</div>
        </div>
    </div>

Here's the markup for those nested grids:

<div class="grid">
        <div class="row">
            <div class="cell half">
                Half
                <div class="grid"><!-- Second level grid -->
                    <div class="row">
                        <div class="cell third">Third</div>
                        <div class="cell two-thirds">
                            Two-Thirds
                            <div class="grid"><!-- Third level grid -->
                                <div class="row">
                                    <div class="cell">Auto</div>
                                    <div class="cell">Auto</div>
                                </div>
                            </div><!-- Third level grid -->
                        </div>
                    </div>
                </div><!-- Second level grid -->
            </div>
            <div class="cell top">Auto Top</div>
            <div class="cell center">Auto Center</div>
            <div class="cell bottom">Auto Bottom</div>
        </div>
    </div>
